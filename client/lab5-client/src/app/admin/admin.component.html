DMCA Takedown Procedure

  1. Check webtech3316@gmail.com for incoming requests.
  2. Determine if there is reasonably sufficient evidence to allow us to identify the collection which has infringed.
  3. By using the searching functionality on admin page, locate and disable the disputed collection
  4. Log the infringement notice and takedown
  5. If the infringing user submits a counter claim, log this is as a disputed collection
  6. Notify the issuer of the initial complaint about counter claim, and ask for further evidence
  7. If the initial issuer does not respond within 14 business days, re-enable the disputed collection and remove it from the disputed log section
